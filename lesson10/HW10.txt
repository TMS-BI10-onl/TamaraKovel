SELECT * FROM Production.WorkOrder
SELECT Every_Year, [1] AS [January], [2] AS [February], [12] AS[December]
FROM 
(SELECT MONTH(DueDate) AS Winter_Month, 
		YEAR(DueDate) AS Every_Year,
		SUM(OrderQty) AS OrderQty
    FROM Production.WorkOrder
	WHERE MONTH(DueDate) = 12 OR MONTH(DueDate) = 1 OR MONTH(DueDate) = 2
	GROUP BY DueDate
) AS SourceTable  
PIVOT
( 
	SUM(OrderQty)
	FOR Winter_Month IN ([1], [2], [12])
) AS PivotTable
ORDER BY Every_Year

Поликлиника
 
1.Пациенты (ID, ФИО, Дата рождения, Адрес, Телефон, ID участка)
2.Врачи (ID, ФИО, Дата рождения, ID отделения, ID участка)
3.Отделения (ID, Название отделения, ID врача)
4.Участки (ID, Номер участка)
5.Нозология (ID, Название, Код по МКБ, ID Отделения)
6.Диагноз (ID, ID пациента, ID нозологии, дата появления первых симптомов, дата обращения, дата выставления диагноза, ID врача)
7.Экстренное извещение (ID, ID пациента, ID диагноза, Дата отправления, ID врача)

Задача 1
Вывести количество пациентов по возрастной структуре (18-30 лет, 31-39 лет, 40-55 лет, 60 и старше), обратившихся в поликлинику с диагнозом "ОРИ" за январь месяц 2021 года, а также за январь месяц 2022 года.

Задача 2
Проверить по всем ли диагнозам отправлены экстренные извещения в положенный срок (1 день после выставления диагноза) за январь-февраль 2022 года.
Вывести ФИО врачей, а также отделения, в случае если не было отправленно экстренное извещение.
